# `seuss`

To arrive at a [universal language](https://en.wikipedia.org/wiki/Universal_language).

## Usage

```console
$ git clone https://github.com/gongahkia/seuss
$ python3 -m venv senv
$ source senv/bin/activate
$ pip install -r requirements.txt
$ python3 src/main.py
```
